# Cognee Knowledge Base Commands

## What Cognee Does
- Ingests documents/data in any format
- Creates searchable knowledge graph (vector + graph search)
- Provides persistent memory for AI agents
- Connects to local llamacpp for embeddings/inference

## How to Use via opencode MCP
- `add this file to my knowledge base`
- `search for information about X`
- `cognify the project`

## How to Use via CLI
- `cognee-cli add "Your text or document content"`
- `cognee-cli cognify`
- `cognee-cli search "Query?"`
- `cognee-cli delete --all`

## Key Features
- Persistent memory across sessions
- Multi-modal (documents, text, code, etc.)
- Graph + Vector search
- Local-only (runs on your machine with local LLM)

## Use Cases
- Store struggles/difficulties you've solved
- Save repetitive processes/workflows
- Build institutional knowledge for future reference
- Create a personal knowledge base for quick retrieval
